üéØ Key Takeaways

Static vs Relative: Both stay in normal flow, but relative can be offset without affecting others.
Absolute vs Fixed: Absolute is relative to a positioned ancestor; fixed is always relative to the viewport.

Pro Tip: When using absolute positioning, always ensure the parent is `position: relative` (or absolute/fixed).
